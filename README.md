# WJD Digital Art Gallery v1.5.3.1 — My Room Profile Fix

แก้บั๊กนักเรียนที่สร้างโปรไฟล์ใหม่ด้วย PIN แล้วกดเมนู "ห้องของฉัน" ไม่ทำงาน

## สาเหตุ
ระบบห้องเดิมผูกกับนักเรียนตัวอย่างที่มีข้อมูลครบ แต่โปรไฟล์นักเรียนที่สร้างใหม่ยังไม่มีข้อมูลห้องส่วนตัว เช่น short, avatar, bio, baseIndex

## สิ่งที่แก้
- สร้างข้อมูลห้องส่วนตัวจากโปรไฟล์นักเรียนอัตโนมัติ
- ให้ห้องของฉันทำงานกับนักเรียนจริงที่สร้างด้วย PIN
- ของตกแต่งและห้องส่วนตัวผูกกับ studentId ของบัญชีจริง
- ไม่แตะ Backend และไม่ต้องอัปเดต Apps Script ใหม่
