# WJD Digital Art Gallery v1.5.7 — Instant Gallery & Quiet Sync

## แนวใหม่
- ไม่มีระบบอนุมัติ
- นักเรียนอัปโหลดแล้วแสดงในแกลเลอรี่ทันที
- นักเรียนลบได้เฉพาะรูปของตัวเอง
- ครูลบรูปได้ทุกคน
- ครูลบยูสเซอร์นักเรียนได้
- ใช้แถบสถานะซิงก์แบบเงียบ ไม่เด้งป๊อบอัพรัว ๆ
