# WJD Digital Art Gallery v1.6.8 — Stable Avatar Rollback

## เหตุผลของเวอร์ชันนี้
ระบบ Avatar ใหม่ 15 ภาพทำให้หน้าโปรไฟล์มีโค้ดซ้อนหลายชั้นและเกิดปัญหากับเมนู “แก้ไขรูปโปรไฟล์”
เวอร์ชันนี้จึงถอยเฉพาะระบบ Avatar กลับไปใช้ชุดเดิม 7 ภาพ เพื่อให้เว็บนิ่งก่อน

## สิ่งที่เก็บไว้
- ระบบห้องของฉันจาก v1.6.6.2
- กดเข้าห้องเพื่อนจากรูป / ชื่อ / การ์ดเพื่อนได้
- ระบบแกลเลอรี ห้อง ของขวัญ แต้ม และภารกิจยังอยู่

## สิ่งที่ถอยกลับ
- ใช้ Avatar เดิม 7 ภาพ
- ตัด/ซ่อนการอัปโหลดรูปโปรไฟล์จากมือถือ
- ตัดระบบ Pixel Avatar 15 ภาพออกจากหน้าเว็บ
- ลดโค้ดซ้อนของระบบโปรไฟล์
