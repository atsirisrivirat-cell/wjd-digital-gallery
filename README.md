# WJD Digital Art Gallery v1.6.9 — Simple Signup & Edit Avatar Later

## แนวคิด
ตอนสมัคร/เข้าใช้งานครั้งแรก นักเรียนไม่ต้องเลือก Avatar
ระบบจะใช้ Avatar เริ่มต้นก่อน แล้วให้นักเรียนไปเปลี่ยนเองภายหลังที่เมนู “แก้ไข Avatar โปรไฟล์”

## สิ่งที่ปรับ
- สมัครง่ายขึ้น: กรอกชื่อ ชั้น ห้อง เลขที่ และ PIN เท่านั้น
- ไม่บังคับเลือก Avatar ตอนสมัคร
- เปิดเมนู “แก้ไข Avatar โปรไฟล์” หลังเข้าสู่ระบบ
- ใช้ Avatar เดิม 7 ภาพ เพื่อความเสถียร
- ปิด/ซ่อนระบบอัปโหลดรูปจากมือถือ
- ตัดผลกระทบจากหน้าต่าง Avatar ทดลองเก่าที่เคยค้าง
- เก็บระบบห้องของฉันและห้องเพื่อนจาก v1.6.6.2
