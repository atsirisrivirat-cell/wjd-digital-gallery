# WJD Digital Art Gallery v1.6.2 — Gift Inventory, UI & Missions Fix

## แก้ไข
- ของขวัญเข้าประวัติแล้ว แต่ไม่เข้าคลังในห้องของฉัน
- ปรับหน้า “ส่งของขวัญ” ให้เป็นระเบียบขึ้น
- เปลี่ยนคำว่า “ของขวัญ&แลกของ” เป็น “ส่งของขวัญ”
- ตัด/ซ่อนส่วนแลกของเดิมจากหน้า UI
- เพิ่มภารกิจศิลป์ใหม่ 10 ภารกิจ

- เมื่อผู้รับล็อกอิน ระบบจะดึง Room State ล่าสุดจาก Google Sheets
- ของขวัญที่ได้รับจะเข้าคลังใน “ห้องของฉัน”
- มีแจ้งเตือนของขวัญใหม่และปุ่มรับทราบ
