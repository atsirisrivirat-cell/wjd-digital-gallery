# WJD Digital Art Gallery v1.5.6 — Image Compression & Smart Sync

## สิ่งที่เพิ่ม
- บีบอัดรูปก่อนอัปโหลดอัตโนมัติ
  - ด้านยาวสุดประมาณ 1200 px
  - JPG quality ประมาณ 72%
  - พยายามลดไฟล์ให้ต่ำกว่า 650 KB
- หน้าอนุมัติครู Smart Sync
  - เข้าเมนูครูแล้วดึงงานจาก Google Sheets อัตโนมัติ
  - เช็กงานใหม่ประมาณทุก 45 วินาที
  - ปุ่มดึงงานยังใช้เป็นรีเฟรชเองได้
- หน้าอนุมัติแยกตัวกรอง
  - รอตรวจ
  - อนุมัติแล้ว
  - ส่งกลับ
  - ทั้งหมด
- Fast Gallery
  - ดึงเฉพาะงาน approved สำหรับแกลเลอรี่
  - หลังอนุมัติแล้วรีเฟรชแกลเลอรี่เร็วขึ้น
- สถานะงานนักเรียนชัดขึ้น

## Backend
ใช้ Backend v1.5.5.1 ได้ต่อ ไม่ต้องอัปเดต Apps Script เพิ่ม
