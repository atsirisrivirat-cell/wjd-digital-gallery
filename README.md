# WJD Digital Art Gallery v1.6.4 — Profile Photo & Avatar Switch

## เพิ่มระบบรูปโปรไฟล์
- นักเรียนอัปโหลดรูปโปรไฟล์จากมือถือได้
- รูปถูกบีบอัดเป็น 320×320 เพื่อโหลดเร็ว
- รูปบันทึกลง Google Drive และข้อมูลบันทึกใน Profiles
- นักเรียนสลับได้ระหว่าง avatar กับรูปที่อัปโหลด
- รูปแสดงข้ามเครื่อง
- ครูรีเซ็ตรูปโปรไฟล์นักเรียนกลับไปใช้ avatar ได้

## ต้องอัปเดต Backend ก่อน
